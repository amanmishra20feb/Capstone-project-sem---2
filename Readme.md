## 🔰 Introduction
The objective of this project is to create a web application that:
- Integrates a public API using fetch()
- Uses array higher-order functions for searching, filtering, and sorting
- Provides a clean and responsive user interface using CSS :contentReference[oaicite:1]{index=1}  

---

## 🌐 Project Idea
**PG Finder Web App**

A web application that allows users to:
- Search PGs based on city
- Filter PGs based on price
- Sort PGs based on price
- Save favorite PGs

---

## 🌐 API Used
- Platform: RapidAPI  
- API: Realty in US API  
- Purpose: Fetch real estate data and display it as PG listings  

---

## 🚀 Features

### 🔍 Search
- Search PGs by city name using filter()

### 🎯 Filtering
- Filter PGs based on price range

### 🔃 Sorting
- Sort PGs by price (Low → High / High → Low)

### ❤️ Favorites
- Save PGs using localStorage

### 📱 Responsive Design
- Works on mobile, tablet, and desktop

### 🎨 UI Design
- Card-based layout
- Animations and hover effects
- Images for better user experience
