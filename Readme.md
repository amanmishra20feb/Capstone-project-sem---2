# 💼 LinkUp – Job Finder Website (LinkedIn Clone)

## 📌 Project Overview

LinkUp is a web-based job finder platform inspired by LinkedIn. It allows users to browse jobs, search and filter listings, view detailed job descriptions, and save jobs for later.

This project demonstrates frontend development skills using HTML, CSS, and JavaScript along with real-time API integration.

---

## 🎯 Objectives

* Build a multi-page responsive website
* Integrate a real-world API for job listings
* Implement search, filter, and sort functionality
* Use Local Storage to persist user data
* Create a LinkedIn-like UI/UX experience

---

## 🚀 Features

### 🏠 Home Page

* User profile preview
* Static feed (posts like LinkedIn)
* Trending jobs section

### 💼 Jobs Page

* Fetches real-time job data using API
* Displays job title, company, and category
* Search functionality
* Filter jobs by type
* Sort jobs alphabetically

### 📄 Job Details Page

* Displays full job description
* Shows selected job information

### ⭐ Saved Jobs Page

* Save jobs using Local Storage
* View saved jobs later

### 👤 Profile Page

* Basic user profile display
* Skills and info section

## ⚙️ How It Works

1. User opens the Jobs page
2. Website fetches job data from API
3. Jobs are displayed dynamically
4. User can:

   * Search jobs
   * Filter jobs
   * Sort jobs
5. Clicking "View" → opens job details page
6. Clicking "Save" → stores job in Local Storage
7. Saved jobs appear in Saved Page
